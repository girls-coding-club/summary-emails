Hi Coders!

Thank you for a phenomenal Girls Coding Club this past Saturday! Some of us learned about Python and R for data analysis, and Javascript for web development. A couple of us worked with PHP and Wordpress. We also discussed web development versus design, Ruby on Rails, and coding for games.

Some of the free resources we are learning from include:

  • Learn Python the Hard Way

    A great book for learning Python for beginners with no experience in programming

  • Eloquent Javascript

    Helpful with the why behind some of what you are told to do on Codeacademy

  • Human Computer Interaction Class on Coursera

    Insight on designing human-centered interfaces on computers

  • Minecraft Coding Class

    Happened in July, will happen again in December
