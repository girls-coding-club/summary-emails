Last week, we learned more about git, the most widely used version control today. We referred to the phenomenal Pro Git Book online a lot.

I remember when I first began programming, I didn't understand the full power of version control. Now, I know that when I really mess something up, or want to experiment with another way of solving a problem, I can switch back and forth between different versions that git keeps track of for me. It is also very important for working with other people on the same code as it helps us track our own versions locally and stay up to date with one central version. The Pro Git Book has a really great introduction to what git and version control are here.

Some other resources about git:

  • https://try.github.io is a great way to try some git commands.

  • Follow along in your command line with gitimmersion.
